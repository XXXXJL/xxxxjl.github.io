接口和继承的区别

| 不同点               | 接口       | 继承         | 补充说明 |
| -------------------- | ---------- | ------------ | -------- |
| 修饰符不同           | Implements | extend       |          |

| 方法是否是抽象方法   | 全部是     | 可是可不是   |          |
| 抽象方法是否必须实现 | 是         | 根据需要决定 |          |
